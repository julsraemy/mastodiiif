IIIF on Mastodon

Follow all your favourite people, projects and institutions active in the International Image Interoperability Framework (IIIF)

If you want to get in touch with people, projects and institutions that use, implement, study and talk about the International Image Interoperability Framework (IIIF - "triple-eye-eff") on Mastodon, this list is for you! It offers an easy method to follow multiple accounts at once. You can can either download the full CSV or decide which account(s) you would like to follow. You can then upload the downloaded CSV in Mastodon.

Here you can find a short instruction, on how to do this.

Futher information on how to be added or removed can be found at the bottom of this page.

Accounts by handle and name with profile link (most recent first)

Full names are only included if they are shown in the profile or on explicit demand. If you want to be on the list, please use this form.

You are interested about IIIF content on mastodon, but you don’t want to be added to the list? Then you could consider following the group @iiif@a.gup.pe.

If you would like to be removed from the list or if there is something that needs to be amended, you can contact me.
